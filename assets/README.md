# 자산 파일 디렉토리

이 디렉토리는 다음과 같은 자산 파일들을 저장하는 곳입니다:

- 이미지 파일 (로고, 아이콘, 배경 이미지 등)
- 인터페이스 요소
- 설명 자료
- 법률 문서 템플릿
- 기타 미디어 자산
