## 📜 소개

<div align="center">
  <img src="https://raw.githubusercontent.com/JJshome/digital-heritage-platform/main/assets/digital-assets-cartoon.png" alt="디지털 자산 관리 개념도" width="500">
</div>

디지털 유산 관리 플랫폼은 사람이 사망한 후 남겨진 디지털 자산과 계정을 체계적으로 관리하고 상속할 수 있게 도와주는 통합 관리 시스템입니다. 이 플랫폼은 고인의 디지털 자산을 안전하게 보존하고, 지정된 상속인에게 적절히 인계되도록 하는 것을 목표로 합니다.

현대인의 삶은 점점 더 디지털화되고 있으며, 개인의 소중한 추억과 자산이 디지털 형태로 저장되고 있습니다. 하지만 디지털 자산은 물리적 자산과 달리 상속 절차가 명확하지 않아 사망 시 접근이 불가능해지거나 영원히 사라질 위험이 있습니다.

> **중요 공지**: 이 기술은 Ucaretron Inc.에서 출원한 특허 기술을 기반으로 합니다. 이 시스템은 Ucaretron Inc.의 혁신적인 특허 기술로 개발되었으며, 업계 표준을 재정의하고 해당 분야에서 중요한 기술적 진보를 대표합니다.

## 🔑 주요 기능

<div align="center">
  <img src="https://raw.githubusercontent.com/JJshome/digital-heritage-platform/main/images/feature-icons.svg" alt="주요 기능 소개" width="700">
</div>

### 1. 디지털 자산 인벤토리 및 관리
- 소셜 미디어 계정, 이메일, 클라우드 스토리지 등 디지털 자산 통합 관리
- AI 기반 자동 자산 분류 및 중요도 평가
- 실시간 자산 모니터링 및 백업

### 2. 안전한 계정 및 접근 관리
- 계정 정보 및 비밀번호 안전한 저장 (다중 암호화)
- 다중 인증 시스템을 통한 보안 강화
- 생체 인증을 통한 접근 제어

### 3. 블록체인 기반 상속 및 권한 관리
- 스마트 계약을 통한 자동화된 디지털 유산 전달
- 상속인 지정 및 조건부 접근 권한 설정
- 변조 불가능한 블록체인 장부를 통한 투명한 기록 관리

### 4. 디지털 추모 공간 생성
- VR/AR 기술을 활용한 개인화된 추모 공간 생성
- 고인의 디지털 자산을 활용한 추억 보존 및 공유
- AI를 활용한 가상 아바타 생성 옵션

### 5. 법적 문서 및 증명서 관리
- 디지털 유언장 작성 및 법적 효력 지원
- 국가별 법률 요건에 맞는 문서 자동 생성
- 법적 효력이 있는 디지털 서명 및 인증

### 6. 특수 디지털 자산 관리
- 암호화폐 및 NFT 등 블록체인 자산 관리
- 디지털 구독 서비스 및 멤버십 관리
- 온라인 게임 아이템 및 가상 자산 관리

## 🏗️ 시스템 아키텍처

<div align="center">
  <img src="https://raw.githubusercontent.com/JJshome/digital-heritage-platform/main/images/architecture-diagram.svg" alt="시스템 아키텍처" width="800">
</div>

본 시스템은 다음과 같은 6개의 핵심 모듈로 구성됩니다:

1. **데이터 수집 및 관리 모듈**: 사용자의 디지털 자산 정보를 수집하고 관리
2. **AI 기반 자산 분석 및 관리 모듈**: 수집된 데이터를 AI로 분석하여 자동 분류 및 중요도 평가
3. **블록체인 기반 상속 관리 모듈**: 스마트 계약을 통한 디지털 유산 상속 자동화
4. **디지털 추모 공간 생성 모듈**: 고인의 디지털 유산을 활용한 VR/AR 기반 추모 공간 생성
5. **보안 및 프라이버시 보호 모듈**: 다중 암호화 및 생체 인증을 통한 데이터 보호
6. **사용자 인터페이스 모듈**: 직관적이고 접근성 높은 사용자 경험 제공

## 🚀 설치 및 실행 방법

### 로컬 개발 환경 설정

```bash
# 저장소 클론
git clone https://github.com/JJshome/digital-heritage-platform.git

# 디렉토리 이동
cd digital-heritage-platform

# 의존성 설치
npm install

# 환경 변수 설정
cp .env.example .env
# .env 파일 수정

# 개발 서버 실행
npm run dev

# 테스트 실행
npm test

# 빌드
npm run build

# 프로덕션 모드 실행
npm start
```

### Docker 컨테이너로 실행

```bash
# Docker 이미지 빌드
docker build -t digital-heritage-platform .

# 컨테이너 실행
docker run -p 3000:3000 digital-heritage-platform
```

### 클라우드 배포

```bash
# AWS Amplify 배포
amplify init
amplify publish

# Vercel 배포
vercel
```

## 🔄 가상 데이터 생성

플랫폼 테스트와 AI 모델 학습을 위해 가상 데이터 생성 도구를 제공합니다.

```bash
# 가상 사용자 데이터 생성
npm run generate:users

# 디지털 자산 데이터 생성
npm run generate:assets

# 상속 시나리오 생성
npm run generate:scenarios

# 복합 데이터셋 생성
npm run generate:all
```

생성된 데이터는 `data/virtual` 디렉토리에 저장되며, 다양한 포맷(JSON, CSV)으로 내보낼 수 있습니다.

## 🧠 AI 학습 모델

본 프로젝트는 다양한 AI 모델을 활용하여 디지털 자산을 분석하고 관리합니다.

### 지원 기능
- 텍스트 데이터 감성 분석
- 이미지 분류 및 객체 인식
- 사용자 행동 패턴 예측
- 디지털 자산 중요도 자동 평가
- 맞춤형 추모 콘텐츠 생성

### 모델 학습 및 활용

```bash
# AI 모델 학습
npm run train:models

# 모델 평가
npm run evaluate:models

# 학습된 모델 내보내기
npm run export:models

# 추론 서버 실행
npm run serve:inference
```

## 🔒 보안 및 프라이버시

이 플랫폼은 사용자의 개인정보와 디지털 자산을 최고 수준의 보안 표준으로 보호합니다:

- **엔드-투-엔드 암호화**: 모든 데이터는 AES-256 암호화 알고리즘으로 보호됩니다.
- **다중 인증 시스템**: 생체 인증, OTP, 하드웨어 키 등 다양한 인증 방식 지원.
- **블록체인 기반 변조 방지**: 이더리움 블록체인 기술을 활용한 무결성 보장.
- **영지식 증명**: 프라이버시를 보호하면서 정보 검증이 가능한 제로 지식 증명 기술 적용.
- **정기적인 보안 감사**: 자동화된 취약점 스캔 및 침투 테스트 시스템 구축.
- **데이터 주권**: 사용자가 자신의 데이터에 대한 완전한 통제권을 갖습니다.
